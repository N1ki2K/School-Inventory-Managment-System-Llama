## 🚀 **How to Use**

1. **Start Backend:**
   ```bash
   cd Development/BE
   pnpm dev
   ```

2. **Start Frontend:**
   ```bash
   cd Development/FE
   pnpm dev
   ```

3. **Open:** `http://localhost:3000`

4. **Try it out:**
   - Click "Inventory" → See all items
   - Try the search bar
   - Filter by status
   - Click "Add Item" button
   - Check out an item
   - Go to "Requests" → Create a request
   - Go to "Reports" → See analytics
   - Go to "Settings" → Explore tabs
